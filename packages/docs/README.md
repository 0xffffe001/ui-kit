# UI Kit docs site

The UI Kit library. A `bootstrap` theme on top of `react-bootstrap`.

### Development

1. run

    ```bash
    npm run dev
    ```

2. visit [localhost:8000](http://localhost:8000)

### Deploying to github pages

1. run

    ```bash
    npm run deploy
    ```

2. visit [hshoff.github.io/ui-kit](https://hshoff.github.io/ui-kit)
